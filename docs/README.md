# Flight Booking Platform - Requirements

This document contains the full requirements for the flight booking platform.

## See req.md for complete requirements

## Project Structure

```
flight-booking-platform/
├── frontend/          # React TypeScript frontend
├── backend/           # Node.js Express backend
├── docs/              # Documentation
└── req.md             # Full requirements document
```

## Quick Start

```bash
# Install dependencies
npm install

# Start development servers
npm run dev
```

## Features

- Flight search and booking
- User authentication and accounts
- Booking management
- Admin dashboard
- Payment integration
- Multi-language support
- Mobile responsive design
